# Mapping Validator (React + Vite + TypeScript)

Интерфейс для строгой проверки маппинга товаров. Нормализация выполняет:

- приведение к нижнему регистру,
- замену пунктуации на пробелы,
- схлопывание пробелов.

Таким образом "Орбит клубничный" не совпадает с "Орбит яблочный".

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173`, загрузите JSON-файл и выберите поля
`expected` и `actual` (по умолчанию `title` и `matched_csv_title`).

## Автозагрузка файла при старте

1. Положите файл в `public/mappings.json`.
2. Приложение загрузит его автоматически при открытии страницы.

Можно указать другой путь параметром `file`, например:

```
http://localhost:5173/?file=/data/airba.json
```
